# Friday AI Assistant

A comprehensive AI-powered virtual assistant inspired by Marvel's FRIDAY, built with Python to provide voice-controlled automation, intelligent conversations, task execution, and productivity assistance through natural language interactions.

The system listens to voice commands, understands user requests, and performs various tasks such as opening applications, searching the web, answering questions, managing system operations, and providing real-time information. Designed to function as a personal desktop assistant, Friday delivers a seamless hands-free experience through speech recognition and AI-powered responses.

## 🚀 Installation

### 1. Clone Repository

```bash
git clone <repository-url>
cd Friday-AI-Assistant
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure Environment Variables

```env
OPENAI_API_KEY=your_api_key
```

### 4. Run Application

```bash
python main.py
```
